TAGS HTML

< h1 >
- Essa tag é usada para definir um cabeçalho de nível 1. Ela geralmente é usada para o título principal de uma página ou seção.
< p >
- Essa tag é usada para definir um parágrafo de texto. É usada para envolver texto que compõe um parágrafo dentro de um documento HTML.
< a >
- Essa tag é usada para criar um link para outra página da web, arquivo ou local no mesmo documento. Ela envolve o texto ou elemento que será clicável para redirecionar o usuário.
< img >
- Essa tag é usada para incorporar uma imagem em uma página da web. Ela requer o atributo src para especificar o caminho para o arquivo de imagem e pode incluir atributos adicionais, como alt para fornecer um texto alternativo para a imagem
